## <a name='WhatisCboard'></a>什麼是 Cboard？

Cboard 是面向有語言和語言障礙的兒童和成人的免費 Web 應用程式，可促進圖片和文字到語音的通訊。

## <a name='Registrationandlogin'></a>註冊和登入

### <a name='HowdoIregisterforCboard'></a>如何註冊 Cboard？

註冊板，請先跟隨歡迎頁面上的步驟，您可以在這裡找到登入選擇。

![看板網站](/images/help/website.png 'Cboard website')

![歡迎頁面](/images/help/welcome-page.png 'Welcome page')

如果您是新使用者，請按 SIGN UP 一個簡短的表單將彈出要求獲取基本資訊以建立您的帳戶。

![板塊註冊](/images/help/signup.png 'Cboard signup')

一旦您填寫了此資訊，您將在收到一封電子郵件之後不久收到一個連結以便確認您的帳戶。

![Email signup](/images/help/signupemail.png 'Email signup')

這種資訊只能由公司掌握，以支援使用板塊，並在必要時提供幫助。 未與其他組織共享(見 [隱私詳情](https://www.cboard.io/privacy/))。

一旦您驗證了您的帳戶，您將可以訪問 Cboard，並將收到另一封確認您已成功驗證您的帳戶的電子郵件。

### <a name='CanIregistermyselfusingmysocialmediaaccounts'></a>我可以使用自己的社交媒體帳戶進行註冊嗎？

是的！在登入頁面， 如果您想要註冊的話，您將被賦予使用您的 Google 或 Facebook 帳戶的許可權。 您也可以用電子郵件地址註冊。

### <a name='HowdoIlogintoCboardonceIamaregistereduser'></a>我是註冊使用者後如何登入 Cboard？

若要登入到板，請按照註冊的第一步登入 但不在歡迎螢幕上選擇 SIGN UP，請選擇 LOGIN。 然後您將被要求提供您最初註冊的電子郵件和您設定的密碼。

![看板登入](/images/help/login.png 'Cboard login')

### 配置板的不同選項是什麼？

一旦您登入到板，您可以訪問不同的選項來配置和開發當板處於解鎖模式時提供的通訊板。 檢視下面所有關鍵選項列表：

![看板設定](/images/help/settings.png 'Cboard settings')

### <a name='HowdoIlogout'></a>如何登出？

您可以透過選擇設定圖示登出，此選項將出現在紅色中。 您也可以在右角選擇鎖定符號，點選三次，一旦您解鎖螢幕和紅色欄出現， 選擇鎖屏左側的圖示 (如果您載入了一張圖片)，登出選項將以紅色顯示。

![看板登出](/images/help/logout.png 'Cboard logout')

### <a name='Whatisthedifferencebetweenaregisteredandanon-registereduser'></a>註冊使用者和未註冊使用者之間有什麼區別？

註冊使用者能夠個性化他們的看板， 和所做的更改將被儲存到他們的帳戶，以便他們在重新登入時可以使用他們的定製看板。 一個未註冊的使用者將無法安全地儲存更改。我們建議只是為了進行登記，所以您的更改將會被儲存。

### <a name='HowdoIupdatemyuserinformation'></a>如何更新我的使用者資訊？

若要更新您的使用者資訊，請按照您試圖登出時使用的相同步驟。 選擇看板右上角的鎖定符號——三次點選或點選，解鎖螢幕後， 選擇鎖左側的圖示 (使用者名稱首字母，照片)，然後您可以編輯您的使用者資訊。

### <a name='HowdoIresetmypassword'></a>如何重置我的密碼？

為了重置您的密碼，請傳送電子郵件 support@cboard.io，我們會回覆您。

## <a name='Features'></a>特徵

### <a name='WhatdevicesandOSaresupported'></a>支援哪些裝置和作業系統？

板電腦可以使用多種裝置線上使用，包括桌面、平板和行動電話，使用 Mac、Windows 和安卓作業系統。 最好在瀏覽器中檢視，如 Chrome、 Firefox、 Edge 和 Safari 等瀏覽器中。Google Play 可以使用安卓應用程式。

![瀏覽器支援](/images/help/browsers.png 'Browser support')

因為板是一個基於 Web 的應用，它將響應單個瀏覽器提供的設定。 這可能意味著您對板提供的設定有不同的結果。例如，您可能會找到不同的語音選項。

別忘了您也可以根據計算機、平板電腦或電話螢幕大小使用瀏覽器放大或縮放。

請詢問板如何為那些有視覺障礙的人使用高對比度模式，因為在使用這種型別的網頁應用時，基於瀏覽器的選項是沒有幫助的。

### <a name='WhichlanguagesaresupportedbyCboard'></a>板支援哪種語言？

Cboard 支援 33 種語言。要選擇不同的語言，請到設定 — 選擇 LANGUAGE 和可用的語言都會出現。 選擇一個選項和“tick”將顯示在語言旁邊。

### <a name='CanCboardreadmymessageoutaloud'></a>Cboard 是否可以朗讀我的訊息？

是的，看板使用您裝置上可用的合成語音輸出來讀取每個符號下面的單獨標籤，因為它們被新增到訊息輸出框中。 您可以選擇一個完整的訊息，它也將被朗讀。 如果聲音的個性化錄音， 單詞或短語已新增到一個符號 (語音-語音錄制下的說明)，您將聽到該錄制而不是合成的語音。 如果翻譯不能用一種語言提供，需要孩子的聲音，或者口音不可接受，這樣做可能非常有用。 請確保您的麥克風已被授予網路訪問許可權。

### <a name='Speechcapabilities'></a>語音能力

為了調整語音能力，進入設定，在 LANGUAGE 類別下是 SPEECH 類別。 當您選擇 SPEECH 時，您將能夠編輯板上的音調和語音速度。

![語音能力](/images/help/speech.png 'Speech capabilities')

### <a name='Exportandimport'></a>匯出和匯入

要導出的兩個選項都在系統分類中的設定中，語言和語音下方。 ExPORT 類別將賦予您上傳 OpenBoard 或 PDF 的選項。 一旦您選擇重置，IMPORT 類別將開啟您儲存的檔案。

![匯出能力](/images/help/export.png 'Export capabilities')

### <a name='Displaycapabilities'></a>顯示能力

顯示設定可以在系統分類中編輯。顯示選項在匯出和匯入下方是正確的。 您只需選擇 DISPLAY，您就可以透過選擇您想要的大小來更改介面大小和應用字型大小。 紅色點應該低於您想要選擇的大小。

![顯示能力](/images/help/display.png 'Display capabilities')

### <a name='CanIchangeanynavigationsettings'></a>我可以更改任何導航設定嗎？

使用 NAVIGATIATION 部分編輯和方便您在看板周圍移動並刪除訊息框中的符號。 設定中 SYSTEM 分類末尾的導航允許您有一個更大的“回退”按鈕，更容易選擇。 也可以允許刪除('x')按鈕在每個符號上方出現，因為它們是在訊息框中顯示的。 這可能使得刪除句子中的符號變得更容易。但同時也會使一些人感到困惑。

### <a name='Navigationthroughtheinterface'></a>透過介面導航

為了編輯和便利導航板，選擇 NAVIGATION, SYSTEM 中的最後一個選單項(所有在設定中)允許您新增某些詳細資訊以便於使用看板。 導航中的第一個選項允許一個較大的“回退”按鈕，因為這個按鈕的大小較容易選擇。 第二個新增‘x’按鈕，更容易刪除符號， 第三個只需要一次點選即可解鎖設定，就更容易解鎖。

![導航能力](/images/help/navigation.png 'Navigation capabilities')

## <a name='Talking'>錄音</a>

### <a name='HowdoIaddapersonallyrecordedvoicetosymbolsonCboard'></a>如何在板上新增個人錄音的聲音？

1. 轉到您要使用的面板。

2. 使用右側的鎖解鎖板。

3. 選擇左側的“編輯”按鈕。

4. 選擇您想要新增語音錄制的符號儲存格或瓷塊。 一旦您選擇了一系列瓷磚或一系列瓷磚，每張都會顯示藍色的墨盒，它們將被突出顯示。

5. 選擇淺藍色欄最右側的 PENCIL 圖示。

6. 磁貼開啟後，您可以選擇使用錄音機新增聲音，單詞或短語等。

7. 新增發聲標籤

8. 最後選擇麥克風符號並進行錄制，您可以錄制多次並可以播放，然後再將其儲存到板上。

   如果選擇了多個符號，請使用 NEXT 轉到下一個符號並重復該過程。

## <a name='BoardEditing'></a>Cboard 編輯

### <a name='HowdoIcreateanemptyboard'></a>如何建立一個空看板？

在看板中，空板等於一個檔案夾。

如果你想要建立一個新的或空的看板，你必須建立一個新的目錄， 並且這將在 Cboard 儲存區域建立一個新的看板(您可以看到您在編輯模式下在中間條上進入 BOARDS 的時間)。

新的空看板將立即用於編輯，可以設定為基本板 (使用建構按鈕)， 這意味著它將是你下次開啟看板時看到的第一個看板。

若要建立一個新檔案夾，請先解鎖右側的螢幕選擇鎖定按鈕。 解鎖螢幕後，在右側選擇+(+)符號以便新增 TILE。 然後您將被賦予新增一個瓷塊/BUTTON 的選項，這是第一個自動選項或選擇 FOLDER 這個動作使您可以新增更多地塊和檔案夾的新進度。

一旦您命名了檔案夾，您就可以以類似方式上傳一張影象來製作帶有顏色設定和標籤的瓷磚。 此新檔案夾將自動連結到它上方的看板(如果包含一行斜體)。 它將作為一個空板出現在您的看板列表中，直到它有符號和標籤。

### <a name='HowdoIpersonalizeanexistingboardinCboard'></a>我如何將現有的板塊個性化？

為了個性化現有的看板， 在看板上方的右上角選擇右上角的 LOCK(帶交叉箭頭下方)。 一旦你做到這一點，另一條條將出現在黑色的 HOME 條下， 並且你會在左側找到一個小塊(EDIT)圖示來新增一個小塊或儲存格。 一旦選擇方塊，灰色條將會轉向藍色， 在正確的手中，你會看到一個小的垃圾桶和一個鉛筆。

![個性化現有看板](/images/help/personalize.png 'personalize an existing board')

為了個性化板，請選擇鉛筆，您將能夠更改標籤，上傳圖片並新增職業化。

### <a name='CanIusemyownpictureswheneditingaboard'></a>編輯板時能使用我自己的圖片嗎？

是的，您可以在編輯面板時使用自己的圖片。 為了使用您自己的圖片， 當您建立一個新的瓷塊時，您可以上傳影象(透過在解鎖頁面後在灰色條上選擇 PLUS SIGN(+)。 您也可以使用自己的圖片編輯現有的瓷磚。

### <a name='CanIchangetheorderingoftheelementsinaboard'></a>我可以更改板上元素的順序嗎？

是的！當你編輯板時，使用 LOCK 符號解鎖螢幕。 然後選擇左側的 EDIT 塊圖示 (低於 BUILD 圖示)。 然後你可以拖動畫圖/符號或檔案夾到你想要把它放在同一個看板中的任何地方。

### <a name='HowdoIselectmultipleelementstoedit'></a>如何選擇要編輯的多個元素？

以同樣的方式更改看板中元素的順序， 但不要選擇和拖動，您可以在畫面/符號正方形中選擇圓圈。 要編輯許多符號，您可以選擇多個符號。 在這樣做之後，在右側選擇 PENCIL 圖示，這將允許您編輯選中的元素。

### <a name='FindSymbols'></a>當我編輯一層時，我如何找到新的符號？

選擇看板欄左側的 EDIT 塊圖示並選擇您想要更改的圖片/符號。 選擇它，一旦藍色環有勾，轉到右邊的 PENCIL 圖示。 在選擇鉛筆後，您會注意到有一個 MAGNIFING GLASS 圖示進行符號搜尋。 當搜尋功能啟動時，您將在編輯欄中看到 SEARCH SYMBOL LIBRY 。 在概念/標籤/單詞中鍵入您正在尋找的並顯示符號。 一旦您選擇了一個符號，它將自動返回編輯 TILE 頁面。

### <a name='FixedBoards'></a>我能把板塊放在固定位置嗎？

By default, when a new board is created, Cboard will add a **fluent board**. Anyway, it is possible to transform the board into a **fixed board**. They have the following features:

- 對於不同的螢幕尺寸或裝置方向，網格不會改變。
- 行和列的數量是可配置的。
- 瓦片將在網格中具有固定位置。

要將板轉換為固定板，只需編輯板，切換工具欄中的選項即可修復。 您的看板將改變外觀，行數和列數將被顯示。

![固定看板](/images/help/fixedBoard.png 'Fixed board')

### <a name='FixedRows'></a>我如何在我的板上設定特定的行數和列數？

First, you have to transform the board into a **fixed board** (see previous topic). In edition mode, your board will display rows and columns indicators. You can interact with these controls in order to increase or decrease the number of rows and columns you want.

## <a name='Scanning'></a>掃描

### <a name='CanIusescanningtechniquestoaccesssymbols'></a>我可以使用掃描技術訪問標志嗎？

## <a name='CommunicatorBuilder'></a>通訊產生器

### <a name='Whatiscommbuilder'></a>通訊建構器是什麼？

通訊產生器是一個功能，允許您處理版面，以便複製、刪除、發布、新增和移除。 為了訪問功能，您需要解鎖設定並點選“建構”按鈕。 你將看到一個新的頁面結構分為三個不同的標籤：板、公共板、我的所有板塊。請參閱下圖：

![通訊產生器](/images/help/communicator.png 'Communicator builder')

### <a name='CommunicatorBoards'></a>通訊板

您將在通訊器產生器中看到的預設標籤是通訊板標籤，它將顯示包含在您通訊器中的所有板塊。 您可以從以下動作按鈕更改您的通訊器：

- 從通訊器中刪除板：此按鈕將退出通訊器中的板，但不會從資料庫中刪除該板，並且在“所有我的板”選項卡下仍然可用。

- 將板設定為根：此按鈕會將板定義為每次登入 Cboard 時顯示的第一個板。

![通訊板](/images/help/communicatorBoards.png 'Communicator boards')

### <a name='PublicBoards'></a>公共板

- 複製看板：此按鈕將建立一個看板的副本，然後將它新增到你的聯絡人中。 因為這是公有板的副本，您可以按自己的要求編輯此板，而不影響原始公共板塊。

![公共板](/images/help/PublicBoards.png 'Public boards')

### <a name='Allmyboards'></a>我的所有看板

請注意，JUST 註冊使用者可以訪問此標籤功能。

- 從通訊器中刪除板：此按鈕將退出通訊器中的板，但不會從資料庫中刪除該板，並且該板仍然可用。

- 將板新增到通訊器：此按鈕將包括板到通訊器。

- 發布板：這個動作將公開發布板，它將顯示在公共板塊下。每個人都可以看到您的板上。
- 取消發布板：此操作將使板塊變為私有，它不會顯示在公共板塊下。沒有人可以看到您的板塊。
- 刪除看板：此操作將刪除看板，無法恢復。看板將從資料庫中刪除。

![我的所有看板](/images/help/AllmyBoards.png 'All my boards')
